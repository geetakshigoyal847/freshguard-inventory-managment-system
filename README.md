FreshGuard Supermarket Inventory Management System

Live Demo:
https://freshguard.rf.gd


Overview

FreshGuard is a PHP and MySQL based supermarket inventory management system designed to help staff manage stock, monitor product expiry, and improve inventory control.


Features

User Authentication
- Staff login system
- New user registration
- Role-based access control
- Session management

Inventory Management
- Manage supermarket stock
- Track product details
- Monitor available inventory levels

Expiry Alert System
- Identify products approaching expiry
- Help reduce food waste
- Improve stock monitoring

Staff Management
- Staff account creation
- Permission-based access
- Admin-controlled access system

Dashboard
- Centralised inventory overview
- Easy access to important system information
- Improved supermarket operations


Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- phpMyAdmin

Installation Setup

1. Download or clone the repository.

2. Create a MySQL database.

3. Import the database.sql file into phpMyAdmin.

4. Update database credentials in:

includes/db.php

5. Run the project using a PHP server.


Project Purpose

FreshGuard helps supermarkets manage inventory efficiently, monitor expiry dates, reduce product wastage, and improve stock management processes.
